Welcome

Morph is an open source cloud transcoding system. It can leverage the scalability of the cloud infrastructure to encode/transcode the video files in fast speed, and dynamically provision the computing resource according to the workload. Morph is implemented in Python, however, it is easy to access the service via RESTful API, command line interface (CLI), and RPC. For the details of the system design, please refer to the report.
